Endzone Analytics is a full-stack application built to explore and visualize data from a hypothetical NFL season. Designed around a robust PostgreSQL database, the project supports querying across 22 custom-built Flask API routes and presents insights through 18 interactive views using AppSmith. The platform enables users to explore over 600 players, 250 games, and 32 teams through a clean, user-friendly interface. Core features include in-depth player stats, weekly matchup outcomes, betting odds, and team-based performance metrics.

Results & Impact

Scalable Architecture

Invested substantial effort into designing a highly relational, normalized schema built for scalability. The data model was structured to be modular, extensible, and easy to evolve.

Containerized

Docker was used to containerize the entire stack, ensuring consistent development, testing, and deployment across all environments and team members.

Collaborative Development

Built by a team of four using Git and GitHub to manage version control. Collaboration included regular meetings, pull request reviews, and pair programming.

User-Centered Features

Every feature was designed around specific user stories such as fans, analysts, coaches, and bettors to ensure the application satisfactorily supported a wide range of use cases.
